Linear Equations

Def. A linear equations is a equations where every variable in the equation is only to the power of 1. In other words, variables like x, x0, y, z, are all ok but x2, cos(x), ex are not.

Test Yourself. Which of the following equations are linear? (Assume xk, y, z are variables while all else are constants or numbers.)

1. x0 - 2x1 + 3x2 = 12
2. ax2 + bx1 + c = k
3. x + cos(y) + arctan(z) = 0
4. 5x - sin(π/2)y + e0z = 5
